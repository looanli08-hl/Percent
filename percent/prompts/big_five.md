你是一位心理学专家。根据以下人格画像，为这个人在 Big Five（大五人格）的五个维度上打分。

五个维度（每个 0-100 分）：
- **Openness（开放性）**：对新体验、创意、知识的开放程度。高分 = 好奇、想象力丰富、喜欢尝试新事物。低分 = 务实、传统、偏好熟悉的事物。
- **Conscientiousness（尽责性）**：自律、条理、目标导向的程度。高分 = 有计划、勤奋、可靠。低分 = 随性、灵活、不拘小节。
- **Extraversion（外向性）**：社交活跃、精力充沛的程度。高分 = 健谈、喜欢社交、爱当焦点。低分 = 安静、独处偏好、深度关系。
- **Agreeableness（宜人性）**：合作、信任、体贴的程度。高分 = 友善、乐于助人、避免冲突。低分 = 直接、竞争性强、质疑他人。
- **Neuroticism（神经质）**：情绪波动、焦虑的程度。高分 = 容易焦虑、情绪化、敏感。低分 = 情绪稳定、冷静、抗压。

输出 JSON 对象，格式：
{{
  "openness": {{"score": 65, "reasoning": "..."}},
  "conscientiousness": {{"score": 72, "reasoning": "..."}},
  "extraversion": {{"score": 58, "reasoning": "..."}},
  "agreeableness": {{"score": 45, "reasoning": "..."}},
  "neuroticism": {{"score": 35, "reasoning": "..."}}
}}

规则：
- reasoning 用中文，1-2 句话解释为什么给这个分数
- 基于画像中的具体证据，不要猜测
- 分数要有区分度，不要全给 50-60 的中间值
- 返回纯 JSON，不要其他文字

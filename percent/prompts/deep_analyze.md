你是一个人格分析师，正在对一份已有的人格发现做深度二轮审视。

你的任务：
1. **发现更深层的模式** — 多个表面发现背后是否有一个更深的特质？
2. **识别矛盾** — 有没有互相矛盾的发现？如果有，这种矛盾本身说明了什么？
3. **补充遗漏** — 现有发现隐含了什么没被明确提取的特质？
4. **调整置信度** — 哪些发现被多个证据支持（应该提高置信度）？哪些只有孤证（应该降低）？

输出 JSON 数组，每个对象：
- "type": "pattern"（深层模式）/ "contradiction"（矛盾）/ "adjustment"（置信度调整）/ "missing"（遗漏补充）
- "content": 用中文描述
- "confidence": 0.0-1.0
- "related_findings": 相关的原始发现编号数组（从1开始）
- "reasoning": 为什么得出这个结论

规则：
- 全部用中文
- 关注"为什么"，不是"是什么" — 表面行为背后的动机
- 矛盾不一定要消除，真实的人本来就矛盾
- 至少输出 5 个深层发现

---

现有发现（{finding_count} 个）：

{findings}

---

Output a JSON array. Return ONLY valid JSON, no other text.

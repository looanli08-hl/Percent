# SHELVED — Percent (formerly Engram)

**Shelved**: 2026-05-03
**Council verdict**: ❌ ABANDON (除非 full rebrand + pivot angle)

## What this was

Open-source Python CLI + Web: imports user's social media data (WeChat, XHS) → builds AI personality model → exports SOUL.md spec. Chat interface ("AI mirror") + developer API. v0.2.0 published on PyPI. XHS launch metrics: 73 likes / 47 saves / 2972 views.

## Council 关键死因

- 🚨 **SOUL.md 名字已被 OpenClaw (Steinberger) 占了 247K stars 生态** — fatal brand collision, Percent 不是该 spec 的 originator
- **AI companion 市场 2025: $120M total, top 33 apps eat 89%** — 极度集中, "mirror yourself" 不是 proven monetization vector outside celebrity clones
- **Custom GPTs 19x growth in 2025** — 免费吃了 DIY clone yourself niche
- Replika €5.6M Italy fine + FTC complaint = privacy 雷区 baseline
- WeChat / XHS export fragile, 经常 break

## What I learned

- **Brand collision check 必须在 naming 之前** — SoulSpec 的 247K 是该 verify 的事
- Companion app 市场 romance/coach indie 在赢 (Chai $30M ARR, Candy.ai $25M ARR), 但 mirror 不是 proven path
- 抖音 + 小红书 内容运营: 双号活人感, 听劝式开发, AIGC 限流对抗 = 国内 platform 现实

## IP that survives

- "AI mirror" 概念归档 — **但不 reuse SOUL.md 名字** (撞 brand)
- 卡片人格分析 + generated 标签系统 实现经验
